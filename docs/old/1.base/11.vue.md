## vue

### 安装 @vue/cli

```
npm install @vue/cli@4.5.0 -g
```

### 安装 yarn, tyarn

```
npm install yarn tyarn -g

```

### 测试 vue, yarn, tyarn

```
tyarn --version
vue --verion
```

## 创建 vue 项目

1. 创建一个文件夹, myprj 用于管理所有的项目
2. 创建项目 项目名称 vue-helloworld
3. 进入项目文件夹 vue-helloworld
4. 运行项目
5. 在浏览器中, 访问 http://localhost:8080 查看页面

```
mkdir myprj
cd myprj
vue create vue-helloworld
cd vue-helloworld
tyarn serve
```
