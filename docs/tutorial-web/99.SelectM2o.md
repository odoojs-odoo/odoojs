### SelectM2o 下拉框

#### SelectM2o 功能

1. 一个组件, 下拉选择框
2. 可搜索
3. 默认显示 7 条, 超过时, 显示 搜索更多
4. 点击搜索更多, 弹出窗口, 进行选择
5. 选中某项时, 触发 on-change 事件
6. 当原始值不为空, 清空搜索框内容时, 触发 on-change 事件
7. on-change 事件的 参数是 value, label

#### SelectM2o props

| 属性          | 说明                                                                 | 类型           | 默认值 |
| ------------- | -------------------------------------------------------------------- | -------------- | ------ |
| value         | 可使用 v-model 双向绑定                                              | String, Number | 无     |
| label         | 初始化, 显示默认 label                                               | String         | 无     |
| limit         | 下拉列表框, 默认显示的条数. <br/>实际选择项超过该值时, 显示搜索更多. | Number         | 无     |
| placeholder   | 选择框默认文字                                                       | String         | 请选择 |
| optionsMethod | 异步搜索的方法 必须设置                                              | Function       | 无     |

#### SelectM2o events

| 事件名    | 说明                                                                              | 默认值       |
| --------- | --------------------------------------------------------------------------------- | ------------ |
| on-change | 选中的 Option 变化时触发. <br/>清空搜索框内容, 并失去焦点, 且原始值不为空时, 触发 | value, label |
